# ssh-mesh resources

`tools.json` is the curated public command catalog for ssh-mesh's own
MCP-compatible methods. Component catalogs for `mesh-init`, `lmesh`, and
`traceweb` live beside those crates and are served or proxied by ssh-mesh.

Keep this file hand-maintained. Do not generate it automatically from code; the
catalog is the intentionally exposed surface for tools and UIs.
